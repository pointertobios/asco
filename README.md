# ASCO

C++20 coroutine based async framework.

## Features

- [x] Async runtime
  - [x] Basic runnable environment
    - [ ] time slice
    - [ ] CFS scheduler
  - [x] Linux full support
  - [ ] Windows full support
  - [ ] MacOS full support
- [ ] Async timer
- [ ] Async channel (pipe)
- [ ] Async IO
  - [ ] Async file read/write
  - [ ] Async TCP/UDP stream
