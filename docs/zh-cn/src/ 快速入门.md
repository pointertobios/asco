# 快速入门

本节将引导您快速入门 ASCO 库的使用。我们将通过一个简单的示例来展示如何使用 ASCO 创建和调度异步任务。

```c++
#include <print>
#include <asco/future.h>

using asco::future, asco::future_spawn;

future<int> bar(int x) {
    std::println("running bar({})", x);
    co_return x;
}

future_spawn<void> foo() {
    std::println("running foo()");
    co_return;
}

future<int> async_main() {
    auto f1 = foo();
    std::println("foo()");
    co_await f1;
    auto f2 = bar(42);
    std::println("bar(42)");
    auto x = co_await f2;
    std::println("bar(42) = {}", x);
    co_return 0;
}
```
